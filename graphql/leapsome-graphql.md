# Leapsome GraphQL API

Leapsome is a people enablement platform for performance management. The API covers performance reviews, OKRs and goals, learning paths, employee surveys, compensation cycles, and engagement analytics.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Leapsome REST API
**Documentation:** https://help.leapsome.com/hc/en-us/articles/9251417820061-Content-API-Access-Reviews-Goals
